## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[Mongodb](https://mongoosejs.com/docs/guide.html) ORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:coverage
```
